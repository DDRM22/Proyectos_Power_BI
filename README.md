# Proyectos_Power_BI
Repositorio de proyectos realizados con Power BI enfocados en la generación de informes

## Tecnologías empleadas

  - Python
  - Power Query
  - DAX

## Guía de uso

  1. Descargamos el archivo `.pbix` y el `.csv`
  2. Abrimos el proyecto en Power BI Desktop
  3. Comprobar que las rutas a los .csv son correctas
  4. Refrescamos los datos para comenzar a visualizar
